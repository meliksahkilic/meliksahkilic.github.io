# Melikşah Kılıç - Original Portfolio

A premium, custom-built portfolio designed specifically for a Data Scientist.

## Design Features
- Custom typography using 'Outfit' and 'JetBrains Mono'
- Modern ambient glow effects and glassmorphism components
- Responsive timeline-based experience section
- Smooth scroll and reveal animations

## Deployment
This site is static and automatically deployed to GitHub Pages via GitHub Actions on every push to `main`.
